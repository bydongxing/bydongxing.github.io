指定考勤类型颜色

::: tip
通过指定 option 属性里的 color 字段可以自定义考勤类型颜色。

颜色值只能是十六进制色值！

指定考勤类型点击事件

::: tip
通过指定 option 属性里的 events 字段可以自定义考勤类型点击事件。
:::

:::demo

:::

只读模式

::: tip
通过设置 readonly 属性为 true 可以设置组件为只读模式，在该模式下不会触发按钮点击事件。

该模式一般用于查看下属考勤场景。
:::

:::demo

:::

Attributes

参数 说明 类型 可选值 默认值
value / v-model 绑定值, 绑定月份 Date — new Date()
show-header 是否显示组件头部 Boolean — true
readonly 是否只读 Boolean — false
data 考勤数据 Object — {}
option 考勤类型配置 Array — option 配置

Events

事件名称 说明 回调参数
month-change 选中值发生变化时触发 目前的选中值（Date）
row-click 当某一行被点击时会触发该事件 day, attendanceData, row, column, event
cell-click 当某个单元格被点击时会触发该事件 day, attendanceData, row, column, cell, event
